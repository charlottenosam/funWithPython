********************* VerySimpleWebBrowser *****************************

    This is a Very Simple Web Browser implemented over Qt and QtWebKit.
    More info on: http://codescience.wordpress.com/2011/04/19/very-simple-web-browser-on-pyqt4-and-webkit/
    
    To Run this program just type:
    
        ~$ python VerySimpleWebBrowser.py
    
    inside the main directory
    
************************************************************************

    Dependencies:
                 - python
                 - pyqt4
